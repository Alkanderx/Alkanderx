ALKANDER INTERACTIVE CHARACTER SHEET

1. Extract the folder anywhere on Windows.
2. Double-click Launch.bat.
3. The sheet opens in your default browser.

All HP, spell slots, resources, inventory and journal changes save automatically in browser local storage.
No installation is required if Python is installed. Without Python, Launch.bat will open index.html directly.

Note: Google Fonts are used when internet is available. The app still works offline with fallback fonts.


This build permanently applies the supplied layout changes while retaining Layout Editor mode.
